# AICA React Starter

Frontend-only React experience built with reusable components and dummy data—no external APIs or console logs. Structure mirrors a professional app layout so real data can drop in later.

## Scripts
- Install: `npm install`
- Develop: `npm start`

## Structure
- `src/api` — mock API surfaces
- `src/assets` — static assets (placeholder)
- `src/components` — `layout`, `features`, `common`, `ui`
- `src/config` — navigation and shared settings
- `src/contexts` — providers (theme)
- `src/data` — dummy data sets
- `src/hooks` — shared hooks
- `src/lib` — tokens/utilities
- `src/pages` — page-level composition (not yet needed)
- `src/services` — mock clients
- `src/styles` — global styles
- `src/utils` — formatters

## Notes
- Dummy data only; swap services with real endpoints when ready.
- Keep components presentation-focused for easy integration later.
